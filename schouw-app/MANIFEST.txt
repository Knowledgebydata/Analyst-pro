DESIGN SPECIFICATIONS
================================================================================

Mobile-First Responsive:
  - Primary: 320px-768px (phones and small tablets)
  - Secondary: 768px+ (tablets and desktops)
  - No fixed widths, flexible layouts
  - Safe area insets for notched devices

Typography:
  - Font family: IBM Plex Sans, -apple-system, BlinkMacSystemFont, Segoe UI
  - Sizes: 12px (labels) → 28px (headers)
  - Weights: 400, 500, 600, 700
  - Line height: 1.5 for body text
  - Letter spacing: 0.5px for labels

Color System (Light Mode):
  - Background: #f8fafc (warm white)
  - Surface: #ffffff (white)
  - Border: #e2e8f0 (light gray)
  - Text: #1e293b (dark slate)
  - Text muted: #64748b (gray)
  - Accent: #10b981 (green)
  - Danger: #dc2626 (red)

Color System (Dark Mode):
  - Background: #0f172a (dark blue)
  - Surface: #1e293b (slate)
  - Border: #475569 (dark gray)
  - Text: #f1f5f9 (light)
  - Text muted: #cbd5e1 (light gray)
  - Accent: #10b981 (green, unchanged)

Component Touch Targets:
  - Buttons: 44x44px minimum
  - Form inputs: 44px height
  - Rating buttons: 44px diameter
  - Toggle switches: 50x28px
  - FAB: 56x56px

Spacing:
  - 8px base unit
  - Padding: 12px, 16px, 24px
  - Margins: 8px, 12px, 16px, 20px
  - Gaps: 8px, 12px

Animations:
  - Transitions: 0.2s for interactions
  - Duration: 0.3s for screen changes
  - Easing: ease, ease-out
  - No blocking animations

DEPLOYMENT REQUIREMENTS
================================================================================

Server Configuration:
  - HTTPS required (service worker needs secure context)
  - MIME types: text/html, application/json, application/javascript
  - CORS headers for external resources
  - Cache headers: short cache for HTML, long cache for assets

Browser Requirements:
  - Chrome 90+ / Edge 90+
  - Firefox 88+
  - Safari 14+ (iOS 14+)
  - Android 6+

APIs Used:
  - Geolocation API (optional)
  - IndexedDB (required)
  - LocalStorage (required)
  - Service Workers (required)
  - Crypto API (required for UUID)
  - Fetch API (required)
